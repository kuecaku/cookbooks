# cookbooks

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve 启动测试环境

npm run dev 启动测试环境,虚拟内存限制为4096

npm run prod 启动生产环境 --执行 .env.prod文件中的node配置
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
